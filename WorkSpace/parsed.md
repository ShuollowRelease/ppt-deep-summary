# Slide 11

**Layout:** DEFAULT

关键技术：TGN时间图网络
TGN（Temporal Graph Networks）核心机制
TGN将动态图表示为带时间戳的事件序列，通过记忆模块、时间编码和图聚合机制支持动态交互场景下的归纳式学习和边预测。
事件序列在时间图中不断更新节点状态
M
记忆模块
为每个节点引入记忆单元，记录历史交互状态
T
时间编码
学习时间依赖关系，捕捉行为时序特征
A
邻域聚合
聚合局部结构信息，学习节点表示
P
边预测
预测节点间交互概率，识别异常边
优势：支持归纳式学习 → 可处理训练阶段未见过的新节点、新边 → 提升对未知攻击模式的识别能力

# Slide 14

**Layout:** DEFAULT

创新特色
围绕APT横向移动“时间、路径、预测”三个问题，将日志事件、图学习和路径推理串成一条检测闭环。
1
连续时间动态图表示
直接利用事件发生的时间顺序进行建模，描述攻击行为在网络空间中的连续演化过程，提升检测及时性与精确性。
2
威胁样本增强机制
结合攻击知识与图结构分析方法，探索基于攻击路径扩展、行为组合和规则生成的样本增强策略。
3
检测和预测协同分析
将异常检测与路径预测结合起来，在发现可疑行为后进一步推断潜在横向移动方向。
从“发现异常”到“判断去向”，让系统不只回答是否有风险，也解释风险如何沿网络扩散。
